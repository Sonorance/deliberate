# Security policy

## Supported versions

Security fixes are provided for the latest published Deliberate release.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use [GitHub private vulnerability reporting](https://github.com/Sonorance/deliberate/security/advisories/new) and include the affected version, impact, reproduction steps, and any suggested mitigation.

Avoid including real project content, credentials, or other sensitive data in a report. Use a minimal synthetic example instead.
