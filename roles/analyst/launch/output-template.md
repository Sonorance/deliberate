# Launch

## Pitch

* **Value proposition:** _one sharp line — who it's for, the problem, the payoff._

## First users

* **Who:** _the single, specific first customer to win (name the persona) — one concise sentence._
* **Why:** _their acute pain and why they're reachable / reference-able — one concise sentence._
* **Who comes next:** _one concise sentence — does this broaden the product's current audience (and how),
  or, if not meaningfully, say so plainly._

## Launch plan

_Sequence the shaped concept into phases — each grounded in a persona + job and the hero journey from
`## shape` (invent no new capabilities; phase only what the concept already implies)._

* **Phase 1 — Hero MVP:** _the core shape + the minimum capabilities that carry the hero user journey
  end-to-end for the first user — enough to validate the customer need and ship fast. What's in, and the
  exact need it validates._
* **Phase 2 — Fast follow:** _the next necessary capabilities that complete the experience and cover the
  adjacent scenarios these users hit — what makes it whole, not merely viable._
* **Phase 3+ — Later:** _the non-necessities — breadth, nice-to-haves, and polish deferred until the core
  need is validated._

## Growth

_Using **only the functionality this case and the concept deliver** (invent no new capabilities): can it
strengthen a **product-led** growth loop? If no/unlikely, say so in one sentence. If yes, name the single
most likely one:_

* **What:** _the loop and how *this feature* drives it._
* **How:** _how to leverage it for growth._

## Key metrics

_All derived from the **telemetry of the capabilities the concept designs** — not abstract
win-conditions. If a growth loop was specified, tie these to it._

* **North Star:** _one key metric that best measures the success of this case after release, readable
  from the shipped feature's telemetry._
* **Guardrail:** _one telemetry guardrail against over-optimizing the North Star._

**Early signals** — at most **two** leading / proxy telemetry metrics that project where the North Star
is heading:
